Full-Stack Digital Marketplace App

A full-stack digital marketplace platform where users can buy, sell, and manage digital products with real-time chat, secure payments, and admin controls.
-----------------------------------------------------------------------------------------------
Features
👤 User Features
🔐 User authentication & authorization
📦 Create, update, and manage listings
🛒 Buy and sell digital products
💬 Real-time chat system between buyers & sellers
📊 View orders and transaction history
💰 Withdraw earnings
🛠️ Admin Features
📋 Manage all listings
✅ Verify user credentials
💳 Monitor transactions & withdrawals
📊 Admin dashboard for analytics

---------------------------------------------------------------------------------------------------

Tech Stack
Frontend
React.js (Vite)
Redux Toolkit (State Management)
Axios (API calls)
CSS / Custom Styling
Backend
Node.js
Express.js
Prisma ORM
Database
(Configured via Prisma – likely PostgreSQL/MySQL)
